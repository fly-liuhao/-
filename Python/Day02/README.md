#### 一、数值类型
1. 整数类型 int
	- 定义：变量名 = 整数
    ```python
    a = 1
    b = -50
    print(type(a),type(b))
    ```
    - 取值范围
    ```python
    Python其实分为整形和长整形
    Python2X 存在整形和长整形：int long 超过32位需要使用long存储
    Python3X 超过32为自动转成长整形存储
    整形的最大值：取决于操作系统
    ```
